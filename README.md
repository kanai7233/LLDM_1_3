# LLDM_1_3
Pàgina de productes

Exercici 3: Pàgina de productes (HTML + CSS)\
Objectiu: Maquetar una pàgina de productes amb estil de taula i elements\
multimèdia.\
Descripció: Heu d'estilitzar la taula de productes perquè sigui llegible i atractiva,\
així com els elements de vídeo i àudio per integrar-los millor a la pàgina.\
Requisits HTML:\
- Mantenir l'estructura de la taula amb <thead>, <tbody>, <th>, <td>.\
- Incloure el vídeo (<video>) i l'àudio (<audio>) amb els seus atributs\
corresponents.\
Requisits CSS:\
- Donar estil a la taula per fer-la llegible: afegir border-collapse, farciment a\
les cel-les (padding) i una línia de separació.\
- Aplicar un estil de franges a les files de la taula (per exemple, canviant el\
color de fons de les files parelles o imparelles amb el selector :nth-child).\
- Afegir un efecte visual a la taula quan l'usuari passa el cursor per sobre\
d'una fila (tr:hover).\
- Estilitzar l'etiqueta <caption> de la taula per centrar-la i posar-la en\
negreta.\
- Afegir un estil per als elements <video> i <audio>, com ara un border, un\
box-shadow o un marge per separar-los del contingut.\
Punts a avaluar:\
- Habilitat per maquetar taules amb CSS de manera professional.\
- Ús de pseudo-classes i pseudo-elements per millorar el disseny de la\
taula.\
- Integració visual d'elements multimèdia.
